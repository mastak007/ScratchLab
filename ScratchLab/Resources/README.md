# Baby Scratch Reference Audio

## Audio Specs
- Format: WAV
- Sample Rate: 44.1 kHz
- Bit Depth: 16-bit
- Normalized: -1dB peak (loudnorm I=-16)
- Silence trimmed

## Development Notes

This folder may contain local development reference audio that is intentionally
not included in shipped app resource phases. Production detection falls back to
the built-in baby-scratch profile when no bundled training library is present.
