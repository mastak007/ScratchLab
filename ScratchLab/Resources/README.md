# Baby Scratch MVP Dataset

## Audio Specs
- Format: WAV
- Sample Rate: 44.1 kHz
- Bit Depth: 16-bit
- Normalized: -1dB peak (loudnorm I=-16)
- Silence trimmed

## Skill Tiers

### reference_pro/ (QBert - Legend)
- 8 samples from "Learn to Scratch" DVD
- Source: qbert_clean_01.wav - qbert_clean_08.wav

### reference_champ/ (CXL - DMC NZ Champion)
- 4 samples, isolated recordings
- Source: cxl_clean_01.wav - cxl_clean_04.wav

### reference_beginner/ (Karl - Learner)
- karl_beginner_01 to 08: Copying CXL style
- karl_qb_09 to 19: Copying QBert style (11 samples)
- Each scratch recorded 2x for variation
